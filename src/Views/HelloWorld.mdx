# Hello Cell Script !

Cell Script is a new designed statically-typed programming language for smart contracts running on UTXO-based chain CKB.

# Example

Here is a sample code used to express **xUDT** on CKB.

```
import "tx"
import "cell"
import "debug"

function main() {

  vector<cell> inputs = tx.inputs();
  vector<cell> outputs = tx.outputs();
  if(inputs.size() < outputs.size()) {
    return false;
  }

  for(cell input: inputs) {
    if(input.capacity < 100) {
      return true;
    }
  }

  if(tx.hasScript("script hash")) { 
    debug.log("find the script hash in tx");
  }

  vector<vector<byte>> witness = tx.witness();
  for(vector<byte> w: witness) {
    debug.log("the witness data is", w);
  }
  
  return true;
}
```
